# MoCA (Anonymous Repository)

## Directory Structure

- `src/`
  - Core code released for anonymous review.
  - Includes: `__init__.py`, `cli.py`, `schemas.py`, `pipeline.py`, `prompts.py`.

- `samples/samples.json`
  - Curated sample set for this anonymous release.

- `sample_assets/affection/`
  - Local media files (image/video/audio) for `affection` samples.

- `sample_assets/intent/`
  - Local media files (image/video/audio) for `intent` samples.

- `sample_assets/stance/`
  - Local media files (image/video/audio) for `stance` samples.
